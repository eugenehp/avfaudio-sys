# avfaudio-sys
Bindings to `AVFAudio/AVFAudio.h` generated with `bindgen`.

Based on https://github.com/augmented-audio/avfaudio-sys

## Supported platforms

- iphoneos (iOS)
- iphonesimulator (iOS Simulator)
- xros (visionos)
- xrsimulator (visionOS Simulator)
- macosx (macOS)
- watchos (watchOS)
- watchsimulator (watchOS Simulator)
- appletvos (tvOS)
